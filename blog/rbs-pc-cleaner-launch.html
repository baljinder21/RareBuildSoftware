RBS PC Cleaner v1.0.0 is out today — a free Windows maintenance utility built on three boring principles: real numbers, reversible changes, and no fake scores. It doesn't scan your registry, doesn't invent a "health score" that sells you a Pro plan, and doesn't delete anything without asking.

I'm Rai, the developer. I'm a solo dev based in Singapore, and the reason this exists is simple: most "free PC cleaner" Google results are either bundled with adware, charge for the actual cleaning step, or are aggressive enough to break things. CCleaner used to be the answer; in the last decade its install bundle and feature paywalls made it harder to recommend. So I built the thing I wanted: clean, free, no ads, never breaks anything you can't undo.

In short: safe junk cleaner, browser + app cache cleaner, disk analyzer, duplicate finder, bulk uninstaller, startup manager, gaming mode, scheduled auto-clean, 9 real theme packs, portable mode. All free. Windows 10/11 only.

Why another PC cleaner?

Most Windows cleaners do one of two unhelpful things:

  1. They run a "registry scan" that reports thousands of imaginary issues to pressure you into buying Pro.
  2. They delete legitimate startup entries or cached data that apps actually need, then leave you to figure out why Discord or Spotify broke.

RBS PC Cleaner takes the opposite approach. Every action is explicit, rated, and reversible. If a clean might cause trouble — say, the Windows Update cache — the tile is labelled Caution and you're told exactly what will happen.

What's inside

  • System Status — real CPU, RAM, disk, uptime and temperature. No scores.
  • Junk Cleaner — temp, recycle bin, Windows Update cache, prefetch, logs — each rated Safe / Caution / Risky.
  • Browser Cleaner — Chrome, Edge, Firefox, Brave, Opera. Cache, cookies, history. Passwords reported, never deleted.
  • App Cache Cleaner — Discord, Microsoft Teams, Spotify, VS Code, Slack, Zoom.
  • Disk Analyzer — read-only folder tree. See what's using your space and drill down.
  • Duplicate Finder — hash-based. Always keeps the oldest file. Auto-skips Windows and Program Files.
  • Bulk Uninstaller — tick programs, uninstall in sequence. Silent mode when supported.
  • Startup Manager — toggle entries. Disabled ones are parked in a sibling key — fully reversible.
  • Gaming Mode — clean + memory trim + High Performance plan + suspend background apps. Resume brings them all back.
  • 9 Theme Packs — real icon packs + 2560×1440 wallpapers + taskbar accent + light/dark. Restore Original reverts exactly.
  • Scheduler — daily or weekly auto-clean via Windows Task Scheduler.
  • Portable Mode — store settings + backups beside the exe. Great for USB sticks.

Safety, spelled out

The rules are simple and global:

  • The app never scans or modifies the Windows registry. There is no registry cleaner feature.
  • A central safety gate blocks deletion inside C:\Windows, Program Files, Program Files (x86), ProgramData, drive roots, and .sys/.dll/.drv/.cat files.
  • Nothing is ever deleted without a confirmation dialog.
  • Duplicate Finder always keeps the oldest file and auto-skips WinSxS, DriverStore, CatRoot and similar.
  • Startup "disable" doesn't delete the entry — it moves it to a sibling Run-Disabled key. One click re-enables.
  • Gaming Mode suspends background apps rather than killing them. The Resume button undoes everything.
  • Before the first theme apply, your original wallpaper, accent and icons are snapshotted so Restore Original reverts exactly.

Install options

Installer (recommended): Run RBS-PC-Cleaner-Setup-1.0.0.exe. Installs to C:\Program Files\RBS Software\RBS PC Cleaner\, adds a Start-menu entry, optional desktop shortcut, and proper uninstaller in Settings → Apps → Installed apps.

Portable ZIP: Extract anywhere and run RBS PC Cleaner.exe. No install, no writes outside the folder. Perfect for USB sticks. SmartScreen may prompt — click More info → Run anyway (the build is unsigned).

What it won't do

We deliberately avoid two things most cleaners do:

  • No registry scan. Modern Windows doesn't have the registry bloat problem of Windows XP. Most "errors" reported by registry cleaners are harmless orphan keys.
  • No patched system DLLs. Our theme engine skins folders, drive icons, file-types, wallpapers and accent — but it refuses to patch the Start button glyph, taskbar tray icons, or signed Microsoft apps. Windows Update would undo the patch and potentially break integrity checks.

System requirements

  • Windows 10 or Windows 11 (64-bit)
  • 150 MB free disk space
  • 2 GB RAM minimum (4 GB recommended)
  • No Python installation required — everything is bundled
  • Administrator rights only for: drive-icon overrides, HKLM startup entries, C:\Windows\Temp cleanup

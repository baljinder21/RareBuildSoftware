Why Is Windows 11 Running Slow? Common Causes

Before jumping into fixes, it helps to understand why Windows 11 slows down in the first place. Most slowdowns have one or more of these root causes:

  • Junk files and temporary data piling up — Windows creates temp files constantly. Over months and years, gigabytes of junk accumulate on your drive.
  • Too many startup programs — Every app you install tends to add itself to startup. Over time you can have dozens of programs loading on boot, each consuming RAM.
  • Outdated drivers — Especially GPU drivers. An outdated graphics driver can cause stuttering, slow UI rendering, and app crashes.
  • Malware or adware — Running silently in the background, consuming CPU, RAM, and network bandwidth without your knowledge.
  • Insufficient RAM — With too many apps open, Windows falls back on slow virtual memory (the page file), creating dramatic slowdowns.
  • HDD instead of SSD — Traditional hard drives are 5–10x slower than SSDs for Windows system operations. If you're still on an HDD, this is your biggest performance bottleneck.
  • Power settings throttling CPU — "Power saver" mode limits processor speed, even when plugged in.
  • Background Windows Update activity — After updates, Windows runs indexing and cleanup tasks that temporarily reduce performance.
  • Fragmented disk (HDD only) — On mechanical drives, file fragmentation significantly increases read times.
  • Overly aggressive antivirus scanning — Some antivirus products run heavy background scans during peak usage hours.

Identifying which of these applies to your situation will help you prioritize which fixes to try first. The 12 methods below address all of these causes.

12 Free Fixes for a Slow Windows 11 PC

Fix 1: Clean Junk Files with Disk Cleanup

Disk Cleanup is Windows' built-in junk file remover. It's been around since Windows 98 and still works well in Windows 11. The key is to use the "Clean up system files" option, which also removes Windows Update leftovers — often the largest category of junk files.

  1. Press Win + R, type cleanmgr and press Enter
  2. Select your system drive (usually C:) and click OK
  3. When the window opens, click "Clean up system files" (requires admin)
  4. Check all boxes in the list — especially "Windows Update Cleanup" and "Temporary Internet Files"
  5. Click OK, then "Delete Files" to confirm

On a PC that hasn't been cleaned recently, this can free up 3–15GB of disk space. Low disk space (under 10% free) on your system drive directly causes Windows to slow down.

Fix 2: Disable Startup Programs

This is arguably the single most impactful fix for slow boot times and high RAM usage at startup. Most users accumulate dozens of startup programs over time without realizing it.

  1. Press Ctrl + Shift + Esc to open Task Manager
  2. Click the Startup apps tab
  3. Sort by "Startup impact" column — focus on High and Medium impact items first
  4. Right-click any non-essential program and select Disable

Safe to disable at startup: Discord, Spotify, Steam, Epic Games Launcher, Adobe Updater, Skype, Teams (if you don't use it at login), OneDrive (optional), Zoom, and any gaming platform launchers. Do not disable antivirus software or system drivers.

Fix 3: Enable Storage Sense

Storage Sense is Windows 11's automatic cleanup feature. Once enabled, it runs on a schedule and automatically removes temporary files, empties the Recycle Bin, and clears downloaded files you've already used.

  1. Open Settings (Win + I)
  2. Go to System → Storage
  3. Toggle Storage Sense to On
  4. Click "Storage Sense" to open its settings and configure it to run monthly
  5. Enable "Delete files in my recycle bin if they have been there for over: 30 days"

Storage Sense won't replace a manual deep clean, but it keeps the system from accumulating junk between your manual cleanups.

Fix 4: Change Power Plan to High Performance

Windows 11 defaults to "Balanced" power mode, which throttles your CPU speed to save power. This is appropriate for battery-powered laptops, but if you're on desktop or a plugged-in laptop, it's leaving performance on the table.

  1. Go to Settings → System → Power & sleep
  2. Click "Additional power settings" (or search "Choose a power plan" in Start)
  3. Select High performance
  4. If you see "Ultimate performance", select that instead for maximum CPU throughput

Important for laptop users: High performance mode significantly increases battery drain. Use it only when plugged in, and switch back to Balanced on battery. The improvement on desktop PCs is noticeable — faster app launches, more responsive UI, better gaming performance.

Fix 5: Update Windows and Drivers

Running outdated Windows builds or drivers can cause performance regressions, security vulnerabilities, and compatibility issues that manifest as slowness or instability.

  1. Go to Settings → Windows Update → Check for updates — install all available updates
  2. Open Device Manager (right-click Start → Device Manager)
  3. Look for any devices with yellow warning icons and update their drivers
  4. For GPU drivers specifically: visit nvidia.com/drivers (NVIDIA), amd.com/support (AMD), or Intel's download center for the latest GPU drivers

GPU driver updates in particular have significant performance impact — manufacturers release optimizations that can improve both gaming and general rendering performance. A driver update alone has been known to resolve stuttering and UI lag issues.

Fix 6: Scan for Malware

Malware is a hidden performance killer. Cryptomining malware, adware, and trojans can silently consume 20–80% of your CPU and significant RAM while running in the background. If your PC is inexplicably slow and Task Manager shows high CPU or memory usage from unknown processes, malware is a real suspect.

  1. Open Windows Security from the Start menu or system tray
  2. Go to Virus & threat protection
  3. Click Scan options and choose Full scan
  4. Click Scan now — allow 15–45 minutes for a thorough scan

Windows Defender is genuinely capable in 2026 — it's no longer the weak tool it once was. For a second opinion, Malwarebytes Free is excellent for catching adware and PUPs (potentially unwanted programs) that traditional antivirus sometimes misses.

Fix 7: Disable Visual Effects

Windows 11's polished animations, transparency, and shadow effects consume CPU and GPU resources. On older hardware (pre-2018 processors, integrated graphics, less than 8GB RAM), these effects can noticeably impact responsiveness.

  1. Search for "Adjust the appearance and performance of Windows" in Start
  2. In the Performance Options dialog, select "Adjust for best performance"
  3. This disables all animations at once
  4. For a compromise: choose "Custom" and re-enable "Smooth edges of screen fonts" and "Show thumbnails instead of icons"

The most resource-intensive effects to disable are window animations, taskbar animations, and the transparency/blur effects. Even on modern hardware, disabling these saves CPU cycles and can reduce DWM (Desktop Window Manager) memory usage.

Fix 8: Check for High-CPU Background Processes

Sometimes a single rogue process is responsible for making your PC feel slow. Task Manager makes it easy to identify and deal with it:

  1. Press Ctrl + Shift + Esc to open Task Manager
  2. Click the Processes tab
  3. Click the CPU column header to sort by CPU usage (highest first)
  4. Identify any process using consistently high CPU when you're not doing anything demanding

Common legitimate culprits: Windows Update (temporary — let it finish), antivirus scans (temporary — wait for completion), Windows Search indexing (temporary after new files added). If you see an unknown process using high CPU continuously, search its name online to determine if it's malware. To end a process: right-click → End task (only for clearly non-system processes).

Fix 9: Increase Virtual Memory

If you have 4GB or 8GB of RAM and regularly run multiple applications simultaneously, you may be hitting your RAM ceiling and forcing Windows to use slow disk-based virtual memory. Configuring the page file properly can help Windows manage this more gracefully.

  1. Search "Adjust the appearance and performance of Windows" in Start
  2. Go to Advanced → Change under Virtual memory
  3. Uncheck "Automatically manage paging file size for all drives"
  4. Select your C: drive, choose "Custom size"
  5. Set Initial size to 1.5× your RAM in MB (e.g., 12,288 for 8GB)
  6. Set Maximum size to 3× your RAM in MB (e.g., 24,576 for 8GB)
  7. Click Set → OK → restart your PC

This gives Windows a predictably-sized page file that doesn't need to dynamically resize, reducing the overhead of memory management.

Fix 10: Defragment Hard Drive (HDD Only)

If you're running Windows 11 on a traditional spinning hard disk drive (HDD), fragmentation is a real performance issue. Over time, files get written in scattered fragments across the disk, forcing the read head to physically seek between locations — significantly slowing read speeds.

Important: Do NOT defragment an SSD. SSDs have no moving parts and are not affected by fragmentation. Defragmenting an SSD provides no benefit and causes unnecessary write cycles that reduce the drive's lifespan.

  1. Search for "Defragment and Optimize Drives" in Start
  2. Select your HDD (it will show "Hard disk drive" in the Media type column)
  3. Click Optimize
  4. Windows also has automatic optimization scheduled — ensure it's enabled

For HDDs that haven't been defragmented in months or years, this can provide significant read speed improvements and better overall system responsiveness.

Fix 11: Perform a Clean Boot

If Windows 11 is slow and you can't identify the cause, a clean boot helps isolate whether third-party software is responsible. A clean boot starts Windows with only Microsoft services and drivers — no third-party software at all.

  1. Press Win + R, type msconfig and press Enter
  2. On the Services tab, check "Hide all Microsoft services", then click Disable all
  3. On the Startup tab, click "Open Task Manager" and disable all startup items
  4. Click OK and restart your PC

If Windows is fast in a clean boot state, a third-party service or startup program is causing the slowdown. Re-enable services in batches to identify the culprit. Remember to reverse these changes (re-enable services and startup items) once you've finished diagnosing — a clean boot is a diagnostic tool, not a permanent configuration.

Windows 11 Slow After an Update — Special Case

If your Windows 11 PC suddenly became slow immediately after a major Windows update (feature update or cumulative update), this is a known and expected temporary condition — not permanent damage.

After major updates, Windows automatically runs several background tasks:

  • Search indexing — re-indexes files for Windows Search, which is CPU and disk-intensive
  • Update cleanup — processes and archives old Windows installation files
  • App migration — updates compatibility layer for installed applications
  • Telemetry and diagnostics — collects and uploads update quality data to Microsoft

These tasks typically complete within 24–48 hours of the update installation. During this period, expect higher CPU usage, slower app launch times, and reduced overall responsiveness. This is normal.

Once the background tasks complete, run Disk Cleanup and select "Clean up system files" — specifically choosing "Previous Windows installation(s)" and "Windows Update Cleanup". These two categories alone can free up 5–20GB of disk space after a major feature update.

Should You Buy a RAM Upgrade?

After applying the 12 free fixes, if your PC is still consistently hitting 80%+ RAM usage during normal use, a hardware upgrade may be warranted. Here's how to think about it:

  • 4GB RAM on Windows 11 — Struggling is almost guaranteed. Try all software fixes first, but a RAM upgrade to 8GB will make a dramatic difference if your motherboard supports it.
  • 8GB RAM — The borderline case. For basic web browsing and office work, 8GB is fine with good optimization. For multitasking with many browser tabs plus applications, you'll frequently hit the ceiling. Try software fixes first — many 8GB users see major improvement after disabling startup bloat.
  • 16GB RAM — The sweet spot in 2026. This is the right amount for most users and shouldn't require a hardware upgrade unless you're doing video editing or running virtual machines.

The bottom line: always try the 12 free fixes before spending money on hardware. Most users find they can dramatically improve performance without any hardware purchase. Only consider a RAM upgrade after confirming that RAM is genuinely the bottleneck (via Task Manager's Performance → Memory tab showing consistently high usage).

Frequently Asked Questions About Slow Windows 11

Why did Windows 11 become slow after an update?

After major Windows updates, the OS performs background tasks like indexing, update cleanup, and app migration that temporarily reduce performance. This usually resolves within 24–48 hours. During this period, avoid restarting repeatedly (which restarts the background tasks). Once the tasks complete, run Disk Cleanup to remove old Windows installation files and free up several GB of disk space — this alone often resolves post-update slowness.

Does factory resetting Windows 11 fix slowness?

A factory reset (fresh install) will make Windows fast again by removing all accumulated junk, bloatware, and software conflicts. However, it removes all your programs and files (unless you choose "Keep my files"). This is a nuclear option. Try all 12 free fixes in this guide first — most users achieve 80% of the performance improvement they'd get from a factory reset, without losing any data or reinstalling applications. Reserve factory reset as a last resort for severely compromised systems.

Frequently Asked Questions

Everything you need to know about RBS and our free Windows tools. Can't find an answer? Contact us →

Is all Rare Build Software really free?▾
Yes — completely and permanently free. No trials, no subscriptions, no hidden upgrades. Download once and use forever on as many computers as you own. We sustain development through optional donations.
Why is Rare Build Software free?▾
We believe useful tools should be accessible to everyone. Too much good software is locked behind paywalls. We sustain development through optional donations and community support — not by charging users.
How do I get future updates?▾
Check this website periodically — we post all new releases here. You can also subscribe to our newsletter to be notified automatically. We don't have an auto-updater to keep the software lightweight.
Can I use it on multiple computers?▾
Yes! All Rare Build Software is freeware. Install and use on as many computers as you personally own or control, completely free. No activation, no license keys, no limits.
Is Rare Build Software safe to use?▾
Absolutely. All Rare Build Software contains no adware, spyware, malware, or bundled unwanted software. Each app does exactly what it says, nothing more. You can verify with any antivirus scanner or submit to VirusTotal.com for 70+ scanner opinions.
My antivirus flagged it — is it a false positive?▾
Yes, this is a common false positive. System tools often trigger antivirus warnings because they perform low-level operations that look similar to what malware does. All Rare Build Software contains zero malware. Submit to VirusTotal.com for a second opinion.
What Windows versions are supported?▾
All Rare Build Software supports Windows 10 (64-bit) and Windows 11 (64-bit). We do not currently support 32-bit systems or older Windows versions like Windows 7 or 8.
What does RBS Optimizer Pro do?▾
RBS Optimizer Pro cleans junk files, manages startup programs, frees up RAM, monitors your system in real time, and lets you run a full one-click optimization. It works completely offline — no internet required.
Does RBS Optimizer Pro require an internet connection?▾
No. All features work completely offline. It doesn't send usage data anywhere, doesn't require an account, and never phones home.
Will it delete important files?▾
No. The junk cleaner only targets safe temporary files, browser caches, Windows update caches, and system logs. It never touches your documents, photos, or downloads folder unless you specifically tell it to.
Do I need administrator rights for Optimizer Pro?▾
Some features require admin rights — the startup manager and deep system cleaning. Windows will prompt you with a UAC dialog when needed. Basic usage works without admin rights.
How do I uninstall RBS Optimizer Pro?▾
Go to Windows Settings → Apps → search for "RBS Optimizer Pro" → click Uninstall. Or use Control Panel → Programs → Uninstall a program. The uninstaller is clean and leaves no hidden files or registry entries.
What is RBS Voice Cloner?▾
RBS Voice Cloner lets you clone any voice from a 5-second audio sample using the XTTS v2 AI engine. You can then generate speech in 28+ languages using that cloned voice — all running locally on your PC.
Does Voice Cloner need an internet connection?▾
Only on first launch to download the ~2 GB AI model from Hugging Face. After that, everything runs 100% offline. No data is ever sent to any server.
Do I need a GPU to use Voice Cloner?▾
No, but it helps. An NVIDIA GPU with CUDA will generate speech much faster. If you don't have one, it falls back to CPU mode automatically — it's slower but still works.
How much disk space does Voice Cloner need?▾
The installer is ~1.4 GB (includes the AI model). We recommend at least 4 GB free disk space and 8 GB RAM for best performance.
Is it legal to clone someone's voice?▾
You should only clone voices you have permission to use — your own voice, or voices where you have explicit consent. Never use voice cloning to impersonate someone or create misleading content. Responsible use is your responsibility.
What's new in RBS Voice Cloner V2 compared to V1?▾
V2 is a major upgrade. The biggest changes: 16 built-in voices (V1 had 8) so you can start generating speech without recording a sample; a 7-band parametric equaliser tuned for voice with 6 presets (Natural, Warm, Bright, Podcast, Phone, Flat); a built-in translator on the Text-to-Speech page that auto-translates text into any of the 17 supported languages; a redesigned Audio Editor with hero waveform, live playback cursor, drag-to-select and right-click cut/crop/copy; and a new EQ-bar logo. V2 also ships with PyTorch + CUDA 12.8 runtime so it works with any modern NVIDIA driver.
Why is RBS Voice Cloner V2 ~2 GB to download?▾
V2 ships with the PyTorch deep-learning runtime + CUDA 12.8 libraries pre-bundled, so you don't need to install Python, PyTorch, or CUDA separately. That's where the bulk of the download goes. The app itself is small — the runtime is the price of being a "double-click and run" GPU app. After install, the XTTS v2 voice model (~2 GB) is downloaded once on first launch.
Do I need an NVIDIA GPU to use Voice Cloner V2?▾
No, but it's much faster with one. With an NVIDIA GPU (RTX 3060 or better) and CUDA 12.x/13.x driver, generation is 5–10× faster than CPU. Without a GPU, the app falls back to CPU automatically — generation still works, it just takes 10–30× longer per sentence. The Diagnose page (Ctrl+D) shows you which mode you're running in and any package issues.
Can I keep using V1 alongside V2?▾
Yes. V1 and V2 install to separate folders and both are still available for download from the website. V1 is smaller (~248 MB) and a good choice if you don't have an NVIDIA GPU and only need basic cloning. V2 is the recommended version for everyone else — bigger download but a much better app.
What languages does V2 support?▾
17 languages, all native to the XTTS v2 engine: English, Spanish, French, German, Italian, Portuguese, Polish, Turkish, Russian, Dutch, Czech, Arabic, Chinese (Simplified), Japanese, Hungarian, Korean, Hindi. The built-in translator (which uses Google Translate, no API key needed but requires internet on first translate) lets you type in any of these and auto-convert to the target language before generating speech.
How do the 6 EQ presets differ?▾
Natural — light cleanup, slight presence + air boost (good default). Warm — bass + low-mid boost for a fuller voice. Bright — high-mid + presence + air boost for clarity. Podcast — broadcast EQ curve (cut sub, lift presence) for spoken-word content. Phone — narrow-band telephony effect for stylised use. Flat — zero everything. Pick a preset to fill the sliders, then click Apply EQ. Reset reverts the audio to its pre-EQ state.
Where are voice profiles and generated audio saved?▾
Generated audio: %APPDATA%\RBS Software\RBS Voice Cloner\output\ by default (configurable in Settings → Storage). Voice profiles (built-in and your own clones): %APPDATA%\RBS Software\RBS Voice Cloner\profiles\. You can export any profile as a single .rbsvoice file from Voice Profiles → Export to share or back up.
What is Life Dashboard?▾
Life Dashboard is an all-in-one personal productivity app with 16 customisable widgets — tasks, habits, finance tracker, weather, calendar, sleep tracker, diet planner, Pomodoro timer, news feed, and more. All data is saved locally on your PC.
Does Life Dashboard need an internet connection?▾
Only the weather and news widgets require internet. Everything else — tasks, habits, finance, calendar, sleep, diet — works completely offline.
Where is my Life Dashboard data stored?▾
All data is stored locally on your computer. Nothing is uploaded to any server. Your personal data stays private and on your machine at all times.
Can I customise the widgets in Life Dashboard?▾
Yes. You can drag widgets to rearrange them, resize them from the bottom-right corner, and show or hide any widget using the Widgets menu. Build your own layout that suits your workflow.
Does RBS PC Cleaner scan or modify the Windows registry?▾
No. RBS PC Cleaner never scans or modifies the registry — there is no registry-cleaner feature, and we will not add one. Modern Windows does not benefit from registry cleaning, and almost every "registry error" reported by other cleaners is a harmless orphan key used to pressure you into a Pro upgrade.
Is everything RBS PC Cleaner does reversible?▾
Nearly all of it. Startup entries are parked in a sibling Run-Disabled registry key (one click to re-enable). Gaming Mode suspends background apps rather than killing them — the Resume button wakes them up exactly where they were. Theme apply snapshots your original wallpaper, accent and icons so Restore Original reverts exactly. Files deleted by the junk and app-cache cleaners go to the Recycle Bin by default (hard delete is opt-in).
What does "Safe / Caution / Risky" mean in the Junk Cleaner?▾
Every clean target is rated so you know what you're clicking. Safe = temp files, recycle bin, browser cache, thumbnails — zero impact if removed. Caution = Windows Update cache, prefetch — safe to remove but may cause small re-fetch or slower first-launch after. Risky = items that could affect app behaviour; only shown if you explicitly enable them in Settings.
Will Duplicate Finder delete anything important?▾
No. Duplicate Finder is hash-based (byte-for-byte comparison), always keeps the oldest file and marks newer copies for deletion, and auto-skips Windows, Program Files, ProgramData, drive roots, WinSxS, DriverStore, and CatRoot. It moves files to the Recycle Bin by default, and every action requires confirmation.
Do the 9 theme packs patch any Windows system files?▾
No. The theme engine only changes wallpaper, accent colour, dark/light mode, folder/drive/file-type icon registry pointers, and copies icons into %APPDATA%. It does not patch explorer.exe, system DLLs, or any signed Microsoft component. That means Windows Update will never undo the theme and won't flag it as integrity corruption. Elements Windows does not allow third-party tools to change (Start button glyph, system-tray icons, Microsoft Store app icons) are left alone by design.
What happens to my saved passwords when the Browser Cleaner runs?▾
Nothing — saved passwords are detected and reported so you know they exist, but never deleted. Browser Cleaner only removes cache, cookies (optional), and history (optional). Bookmarks, autofill form data, and extensions are not touched.
Do I need admin rights to use RBS PC Cleaner?▾
No, for most actions. Admin is only requested when you do something system-wide: overriding drive icons, toggling HKLM (machine-scope) startup entries, cleaning C:\Windows\Temp, uninstalling programs that themselves need elevation, or creating system-wide scheduled clean tasks. The app prompts "Relaunch as admin" at the right moment — it does not force elevation up front.
Can I run RBS PC Cleaner from a USB stick?▾
Yes. Use the portable ZIP (not the installer), extract the RBS PC Cleaner folder anywhere you like, and run RBS PC Cleaner.exe. Turn on Portable Mode in Settings to store config and theme backups beside the exe instead of in %APPDATA%. Keep the _internal folder next to the exe — do not move just the exe.
SmartScreen warns "unrecognised app". Is it safe?▾
Yes. The build is unsigned (code-signing certificates are expensive for a free-forever project) so Windows SmartScreen shows its standard warning on first launch. Click More info → Run anyway. You can verify the build by scanning on VirusTotal.com — the app contains no malware, adware, or telemetry.
What Windows versions does RBS PC Cleaner support?▾
Windows 10 and Windows 11, 64-bit only. It does not require Python, .NET, or any runtime installed — everything is bundled inside the app. 32-bit Windows and older versions (7, 8, 8.1) are not supported.

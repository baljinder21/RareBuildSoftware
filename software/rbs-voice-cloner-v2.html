Key Features

🎤

16 Built-in Voices + Unlimited Clones

6 male, 6 female, 2 boy, 2 girl ready to go. Plus clone any voice from a 30-second sample and save unlimited custom voice profiles.

🌐

17 Languages + Auto-Translate

Native XTTS v2 generation in English, Spanish, French, German, Italian, Portuguese, Polish, Turkish, Russian, Dutch, Czech, Arabic, Chinese, Japanese, Hungarian, Korean, Hindi. Translator built into the TTS page.

🎚️

7-Band Parametric Equaliser

Voice-tuned EQ (Sub 60 Hz / Bass 200 Hz / Low Mid 500 Hz / Mid 1.5 kHz / Hi Mid 3.5 kHz / Presence 8 kHz / Air 12 kHz) with 6 presets — Natural, Warm, Bright, Podcast, Phone, Flat.

✂️

Redesigned Audio Editor

Hero waveform with live playback cursor, drag-to-select with right-click Cut / Delete / Crop / Copy, tabbed effects (Quick · Adjust · Trim & Fade · Equaliser), and full keyboard shortcuts.

⚡

GPU Accelerated (CUDA 12.x / 13.x)

Ships PyTorch with CUDA 12.8 runtime — 5–10× faster than CPU. Falls back to CPU automatically if no CUDA GPU is present. Diagnose page (Ctrl+D) shows GPU vs CPU mode.

🔒

Fully Offline After Setup

XTTS v2 model loads locally, no telemetry, no cloud. Internet only needed for the one-time ~2 GB model download and the optional translator.

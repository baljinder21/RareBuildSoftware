Key Features

🧽

Safe Junk Cleaner

Temp, recycle bin, Windows Update cache, prefetch, logs — each rated Safe / Caution / Risky, nothing deleted without confirmation.

🌐

Browser Cleaner

Clears cache, cookies, history for Chrome, Edge, Firefox, Brave, Opera. Saved passwords are reported but never deleted.

📦

App Cache Cleaner

Wipes Discord, Microsoft Teams, Spotify, VS Code, Slack, Zoom cache in one click.

📊

Disk Analyzer

Read-only folder tree — see exactly what is using your space and drill down click by click.

🔍

Duplicate Finder

Hash-based duplicate detection, always keeps the oldest file, auto-skips Windows and Program Files.

🗑️

Bulk Uninstaller

Tick multiple installed programs and uninstall in sequence — silent mode when supported.

🚀

Startup Manager

Toggle any startup entry on/off. Disabled entries are parked in a sibling key — fully reversible.

🎮

Gaming Mode

One click: safe clean + memory trim + High-Performance power plan + suspend background apps. Resume brings everything back.

🎨

9 Theme Packs

Real icon packs + matching 2560×1440 wallpapers + taskbar accent. Reversible with one click — Restore Original.

⏰

Scheduler

Daily or weekly silent auto-clean via Windows Task Scheduler.

🛡️

Safety Gate

Blocks deletion inside C:\Windows, Program Files, ProgramData, drive roots — and never scans the registry.

💼

Portable Mode

Store settings + backups next to the exe — perfect for USB sticks and IT technicians.
